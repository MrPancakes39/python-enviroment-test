# Test

## Quick Setup

Open Powershell in Windows:

```powershell
> python -m venv venv
> .\venv\Scripts\Activate.ps1
> pip install -r .\requirements.txt
> python .\main.py # run program
```

In Linux (using Bash):

```shell
$ python3 -m venv venv
$ ./venv/bin/activate
$ pip install -r ./requirements.txt
$ python3 ./main.py # run program
```
